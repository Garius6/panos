# Ветвления и циклы

## `если` — выражение

```panos
пер результат = если x > 0 тогда
    "плюс"
иначе
    "ноль или минус"
конец
```

Обе ветки должны быть совместимого типа. Ветка `если` без `иначе` имеет
тип `Пусто`.

## `пока`

```panos
пер i = 0
пока i < 10 цикл
    i = i + 1

    если i == 3 тогда
        продолжить
    конец

    если i > 7 тогда
        прервать
    конец
конец
```

## `для` — три формы

По элементам (нужны `.длина()` и `[индекс]`, как у массива):

```panos
для элемент в массив(1, 2, 3) цикл
    // ...
конец
```

По записям карты (карта сама не индексируется — только через `.записи()`):

```panos
для (ключ, значение) в карта.записи() цикл
    // ...
конец
```

Числовой диапазон, включительно с обеих сторон:

```panos
для сч = 0 по 4 цикл
    // 0, 1, 2, 3, 4
конец
```

`для` — сахар над `пока`, `продолжить`/`прервать` работают одинаково во
всех формах.

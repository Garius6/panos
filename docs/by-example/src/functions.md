# Функции

```panos
функ сложить(a: Число, b: Число) -> Число
    a + b
конец
```

Типы параметров и результата обязательны всегда, включая `-> Пусто` для
функций без осмысленного результата.

## Явный возврат

```panos
функ знак(x: Число) -> Строка
    если x < 0 тогда
        возврат "отрицательное"
    конец
    "неотрицательное"
конец
```

## Функции как значения

```panos
функ применить(ф: функ(Число) -> Число, значение: Число) -> Число
    ф(значение)
конец
```

## Лямбды

```panos
функ старт() -> Число
    применить(функ(x)
        x + 1
    конец, 41)
конец
```

Типы параметров лямбды можно опустить, если компилятор выводит их из
контекста (как здесь — из типа параметра `ф` у `применить`). Если контекста
нет, аннотируйте явно: `функ(x: Число) ... конец`.

# Hello world

```panos
функ старт() -> Число
    10 + 20
конец
```

Результат последнего выражения `старт()` печатается автоматически — вывод:

```text
30
```

Явная печать — через `ввод_вывод` (нужен `импорт`, недоступен в браузерной
песочнице):

```panos
импорт ввод_вывод

функ старт() -> Пусто
    ввод_вывод.печать("привет, panos")
конец
```

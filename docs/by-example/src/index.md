# panos by examples

Сборник рабочих сниппетов по panos, по одной теме на страницу. В отличие
от [тура](https://garius6.github.io/panos/tour/) — не читать по порядку,
а искать нужную тему. В отличие от [основной документации](https://garius6.github.io/panos/) —
минимум прозы, код с комментариями вместо объяснений.

Все примеры рабочие — можно вставить в [веб-песочницу](https://garius6.github.io/panos/playground/)
как есть.

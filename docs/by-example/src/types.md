# Типы и операторы

## Арифметика и сравнение

```panos
пер a = 10 + 5 * 2      // 20 — обычный приоритет
пер b = a > 12          // истина
пер c = a <> 12         // "не равно" — <>, не !=
```

`+ - * /` — числа, `+` дополнительно конкатенирует строки. Нет `%` и
`**`/`^` как операторов (см. `математика.степень` в стандартной библиотеке
основной документации).

## Логика

```panos
пер a = истина и ложь
пер b = ложь или истина
пер c = не ложь
```

Слова, не `&&`/`||`/`!`.

## Строки

```panos
пер s = "ключ = \"значение\"\n"   // \n \t \r \" \\
пер буква = "panos"[0]            // "p" — индексирование даёт Строку
```

## Тупы

```panos
пер точка = (10, 25)
пер x = точка.0
пер y = точка.1
```

## Имена составных типов

Для справки — разбор каждого дальше в туре/документации:

```text
(Число, Строка)               — тупл
функ(Число) -> Булево          — тип функции
Массив(Число)                  — массив
Соответствие(Строка, Число)    — карта
Опция(Число)                   — опциональное значение
Результат(Число, Ошибка)       — результат с ошибкой
```

# Переменные

```panos
пер x = 10                    // тип выведен: Число
пер имя: Строка = "panos"     // тип явный

x = x + 1                     // присваивание, без пер
```

`пер` — единственная форма объявления, нет отдельных `var`/`const`.
Переменная не может иметь тип `Пусто`:

```panos
функ ничего() -> Пусто
конец

функ старт() -> Число
    пер x = ничего()   // Type Error: переменная 'x' не может иметь тип 'Пусто'
    0
конец
```

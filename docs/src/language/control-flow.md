# Управляющие конструкции

## If как выражение

```panos
если x > 0 тогда
    "плюс"
иначе
    "ноль или минус"
конец
```

`если` — выражение, не оператор. Если есть `иначе`, обе ветки должны иметь
совместимые типы (веткa с `паника(...)` имеет тип `Никогда` и совместима с
любой другой веткой). `если` без `иначе` имеет тип `Пусто`.

## `пока`

```panos
пер i = 0
пока i < 10 цикл
    i = i + 1
конец
```

`пока` используется ради побочных эффектов и сам по себе имеет тип `Пусто`.
Внутри — `продолжить` (перейти к следующей проверке условия) и `прервать`
(выйти из цикла):

```panos
пер сумма = 0
пер i = 0

пока i < 10 цикл
    i = i + 1

    если i == 3 тогда
        продолжить
    конец

    если i > 5 тогда
        прервать
    конец

    сумма = сумма + i
конец
```

## `для`

`для` — синтаксический сахар над `пока` (разворачивается на этапе разбора,
никакой отдельной сущности в рантайме нет). Три формы.

По элементам итерируемого значения (массив, либо `.записи()` карты):

```panos
для элемент в массив(1, 2, 3) цикл
    ввод_вывод.печать(элемент)
конец

для (ключ, значение) в карта.записи() цикл
    ввод_вывод.печать(ключ)
конец
```

Числовой диапазон, границы включительны с обеих сторон:

```panos
для сч = 0 по 4 цикл
    ввод_вывод.печать(сч)
конец
```

переберёт `0, 1, 2, 3, 4`. `продолжить`/`прервать` работают внутри `для`
так же, как внутри `пока`.

## Массивы и карты (предварительно)

В примерах выше уже встречались `массив(...)` и `.записи()` — полноразбор
типов `Массив`/`Соответствие` и их методов будет в отдельной главе; здесь
достаточно знать, что `для X в Y цикл` работает с любым значением, у
которого есть `.длина()` и `[индекс]` (массив), либо явно вызван
`.записи()` (карта, которая сама по себе не индексируется позиционно).

# Управляющие конструкции

## If как выражение

```panos
функ старт() -> Строка
    пер x = 5
    если x > 0 тогда
        "плюс"
    иначе
        "ноль или минус"
    конец
конец
```

`если` — выражение, не оператор. Если есть `иначе`, обе ветки должны иметь
совместимые типы (веткa с `паника(...)` имеет тип `Никогда` и совместима с
любой другой веткой). `если` без `иначе` имеет тип `Пусто`.

## `пока`

```panos
функ старт() -> Число
    пер сумма = 0
    пер i = 0

    пока i < 10 цикл
        i = i + 1

        если i == 3 тогда
            продолжить
        конец

        если i > 7 тогда
            прервать
        конец

        сумма = сумма + i
    конец

    сумма
конец
```

`пока` используется ради побочных эффектов и сам по себе имеет тип
`Пусто`. Внутри — `продолжить` (перейти к следующей проверке условия) и
`прервать` (выйти из цикла).

## `для`

`для` — синтаксический сахар над `пока` (разворачивается на этапе разбора,
никакой отдельной сущности в рантайме нет). Три формы.

По элементам итерируемого значения:

```panos
функ старт() -> Число
    пер сумма = 0
    для элемент в массив(1, 2, 3) цикл
        сумма = сумма + элемент
    конец
    сумма
конец
```

По записям карты (карта сама не индексируется — только через `.записи()`):

```panos
функ старт() -> Число
    пер карта: Соответствие(Строка, Число) = соответствие(
        "a" = 1,
        "b" = 2,
    )
    пер сумма = 0
    для (ключ, значение) в карта.записи() цикл
        сумма = сумма + значение
    конец
    сумма
конец
```

Числовой диапазон, границы включительны с обеих сторон:

```panos
функ старт() -> Целое
    пер сумма: Целое = 0
    для сч = 0 по 4 цикл
        сумма = сумма + сч
    конец
    сумма
конец
```

`0 + 1 + 2 + 3 + 4 = 10`. `продолжить`/`прервать` работают внутри `для`
так же, как внутри `пока`. Счётчик `сч` — всегда `Целое` (см. [«Базовые
типы» → Число и Целое](./basic-types.md#число-и-целое)), поэтому
`сумма` тоже должна быть `Целое` — без явной аннотации она вывелась бы
как `Число` из литерала `0` и не совпала бы с `сч` в `сумма + сч`.

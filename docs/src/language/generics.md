# Дженерики

## Generic-структуры

```panos
тип Коробка[T] = структура
    значение: T
конец

реализация Коробка
    функ получить(это: Коробка) -> T
        это.значение
    конец
конец

функ старт() -> Число
    пер к = Коробка(42)
    к.получить()
конец
```

Тип-параметр в квадратных скобках после имени. Конструктор выводит `T` из
переданного аргумента — явно писать `Коробка(Число)` можно, но обычно не
нужно.

## Метод со своим type-параметром

Метод может иметь собственный тип-параметр, отдельный от `T` владельца
(нужно, например, чтобы обернуть значение другого типа):

```panos
тип Коробка[T] = структура
    значение: T
конец

реализация Коробка
    функ обернуть[U](это: Коробка, x: U) -> U
        x
    конец
конец

функ старт() -> Строка
    пер к = Коробка(42)
    к.обернуть("текст")
конец
```

## Generic-функции

```panos
функ тождество[T](x: T) -> T
    x
конец

функ первый_элемент[T](xs: Массив(T)) -> T
    xs[0]
конец

функ старт() -> Число
    первый_элемент(массив(тождество(7), 8, 9))
конец
```

## Generic-структура, реализующая обычный интерфейс

Реализовать интерфейс может и generic-тип — интерфейс тут ничего не знает
про `T`, описывает методы без получателя как обычно:

```panos
тип Печатаемый = интерфейс
    функ вСтроку() -> Строка
конец

тип Коробка[T] = структура
    значение: T
конец

реализация Печатаемый для Коробка
    функ вСтроку(это: Коробка) -> Строка
        "коробка"
    конец
конец

функ показать(x: Печатаемый) -> Строка
    x.вСтроку()
конец

функ старт() -> Строка
    показать(Коробка(5))
конец
```

Если параметр типа нужен самому интерфейсу (а не только реализующей
структуре) — это отдельная возможность, generic-интерфейсы, см.
[«Интерфейсы» → generic-интерфейсы](./interfaces.md#generic-интерфейсы).

## Рекурсивные generic-перечисления

ADT может ссылаться сам на себя внутри собственного варианта:

```panos
тип Дерево[T] = перечисление
    Лист(T)
    Узел(Дерево(T), T, Дерево(T))
конец

реализация Дерево
    функ корень(это: Дерево) -> T
        выбор это
            Лист(x) -> x
            Узел(_, x, _) -> x
        конец
    конец
конец

функ старт() -> Число
    пер дерево = Дерево.Узел(Дерево.Лист(1), 7, Дерево.Лист(3))
    дерево.корень()
конец
```

# Интерфейсы

```panos
тип Атакующий = интерфейс
    функ атаковать(урон: Число) -> Число
конец

тип Игрок = структура
    здоровье: Число
конец

реализация Атакующий для Игрок
    функ атаковать(это: Игрок, урон: Число) -> Число
        это.здоровье - урон
    конец
конец

функ старт() -> Число
    пер герой = Игрок(100)
    пер атакующий: Атакующий = герой
    атакующий.атаковать(15)
конец
```

`реализация Интерфейс для Тип` — сигнатуры методов в интерфейсе не
включают получателя (он неявный, добавляется в реализации первым
параметром). Типизация номинальная, не структурная: тип реализует
интерфейс только через явный блок `реализация ... для ...`, случайное
совпадение сигнатур методов не считается.

Интерфейсный тип у переменной/параметра даёт динамическую диспетчеризацию
— какой конкретно метод вызовется, решается по фактическому значению в
рантайме:

```panos
тип Фигура = интерфейс
    функ площадь() -> Число
конец

тип Круг = структура
    радиус: Число
конец

реализация Фигура для Круг
    функ площадь(это: Круг) -> Число
        это.радиус * это.радиус * 314 / 100
    конец
конец

тип Квадрат = структура
    сторона: Число
конец

реализация Фигура для Квадрат
    функ площадь(это: Квадрат) -> Число
        это.сторона * это.сторона
    конец
конец

функ общая_площадь(a: Фигура, b: Фигура) -> Число
    a.площадь() + b.площадь()
конец

функ старт() -> Число
    общая_площадь(Круг(10), Квадрат(4))
конец
```

# Опция и Результат

Из [прелюдии](./modules.md) — доступны в любом файле без `импорт`.

## `Опция(T)`

```panos
функ найти_бонус(есть: Булево) -> Опция(Число)
    если есть тогда
        Опция.Есть(7)
    иначе
        Опция.Нет()
    конец
конец

функ старт() -> Число
    найти_бонус(истина).получить(0)
конец
```

Методы: `.есть()`, `.пусто()`, `.значение()` (паникует на `Нет`),
`.получить(запасное)`, `.запас(другая)`, `.ожидать(сообщение)` (паникует с
этим сообщением на `Нет`), `.результат_или(ошибка)` (конвертация в
`Результат`), `.заменить_значение(новое)`.

## `Результат(T, E)`

```panos
функ поделить(a: Число, b: Число) -> Результат(Число, Ошибка)
    если b == 0 тогда
        Результат.Неудача(Ошибка("деление", "деление на ноль"))
    иначе
        Результат.Успех(a / b)
    конец
конец

функ старт() -> Число
    поделить(10, 2).получить(0)
конец
```

`Ошибка(код, сообщение)` — встроенный тип (не нужно объявлять свой), поля
`.код`/`.сообщение`. Методы `Результат`: `.успех()`, `.ошибка()`,
`.значение()` (паникует на `Неудача`), `.причина()` (паникует на `Успех`),
`.получить(запасное)`, `.получить_ошибку(запасное)`, `.запас(другой)`,
`.ожидать(сообщение)`, `.ожидать_ошибку(сообщение)`, `.опция()`/
`.ошибка_опция()` (конвертация в `Опция`), `.заменить_значение(новое)`,
`.заменить_ошибку(новая)`.

## Оператор `?`

Ранний выход: если операнд — `Нет`/`Неудача`, функция сразу возвращает
его; иначе `?` разворачивает значение и выполнение продолжается.

```panos
функ найти_бонус(имя: Строка) -> Опция(Число)
    Опция.Есть(7)
конец

функ бонус_с_прибавкой(имя: Строка) -> Опция(Число)
    Опция.Есть(найти_бонус(имя)? + 1)
конец

функ старт() -> Число
    бонус_с_прибавкой("игрок").получить(0)
конец
```

`?` работает и с `Результат` — сама функция, в теле которой стоит `?`,
должна возвращать совместимый `Опция`/`Результат`.

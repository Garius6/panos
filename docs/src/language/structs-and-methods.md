# Структуры и методы

## Объявление и конструктор

```panos
тип Точка = структура
    x: Число
    y: Число
конец

функ старт() -> Число
    пер p = Точка(10, 20)
    p.x + p.y
конец
```

Конструктор — вызов по имени типа, аргументы в порядке объявления полей.
Поле-тупл, поле-структура и т.п. — обычные типы, ничего особого:

```panos
тип Игрок = структура
    имя: Строка
    координаты: (Число, Число)
конец

функ старт() -> Число
    пер игрок = Игрок("Гайдар", (1, 2))
    игрок.координаты.0
конец
```

## Методы

Объявляются в блоке `реализация Тип`. Первый параметр метода — получатель,
его тип должен совпадать с типом владельца; имя параметра — обычное имя,
никакого зарезервированного `self`/`this`, идиоматично называть `это`:

```panos
тип Счётчик = структура
    значение: Число
конец

реализация Счётчик
    функ увеличить(это: Счётчик, шаг: Число) -> Число
        это.значение + шаг
    конец
конец

функ старт() -> Число
    пер c = Счётчик(10)
    c.увеличить(5)
конец
```

Структуры неизменяемы через методы напрямую — `увеличить` выше не меняет
`c`, а возвращает новое значение. Присваивание полю целиком переопределяет
переменную:

```panos
тип Счётчик = структура
    значение: Число
конец

реализация Счётчик
    функ увеличить(это: Счётчик, шаг: Число) -> Счётчик
        Счётчик(это.значение + шаг)
    конец
конец

функ старт() -> Число
    пер c = Счётчик(10)
    c = c.увеличить(5)
    c.значение
конец
```

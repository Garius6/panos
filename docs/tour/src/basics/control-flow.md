# Ветвления и циклы

`если` — выражение, не оператор, поэтому у него есть значение:

```panos
пер знак = если x > 0 тогда
    "плюс"
иначе
    "ноль или минус"
конец
```

`пока` — обычный цикл с условием, `продолжить`/`прервать` работают как
везде:

```panos
пер i = 0
пока i < 10 цикл
    i = i + 1
    если i == 3 тогда
        продолжить
    конец
    если i > 7 тогда
        прервать
    конец
конец
```

`для` — сахар над `пока`, три формы:

```panos
для элемент в массив(1, 2, 3) цикл
    // ...
конец

для (ключ, значение) в карта.записи() цикл
    // ...
конец

для сч = 0 по 4 цикл
    // включительно с обеих сторон: 0, 1, 2, 3, 4
конец
```

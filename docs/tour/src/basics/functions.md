# Функции

```panos
функ сложить(a: Число, b: Число) -> Число
    a + b
конец
```

Типы параметров и результата всегда указываются явно — даже когда функция
ничего не возвращает (тогда пишут `-> Пусто`). Значение последнего
выражения тела и есть результат, `возврат` нужен только для раннего
выхода:

```panos
функ знак(x: Число) -> Строка
    если x < 0 тогда
        возврат "отрицательное"
    конец
    "неотрицательное"
конец
```

Функции — значения: их можно передавать как аргументы. Лямбда — это
`функ(...) ... конец` без имени, типы параметров можно опустить, если
компилятор способен их вывести из контекста:

```panos
функ применить(ф: функ(Число) -> Число, значение: Число) -> Число
    ф(значение)
конец

функ старт() -> Число
    применить(функ(x)
        x + 1
    конец, 41)
конец
```

# Hello, world

Точка входа программы — функция `старт`. Значение её последнего выражения
и есть результат программы.

```panos
функ старт() -> Число
    10 + 20
конец
```

Запустите — увидите `30`. Никакого `печать` не нужно: значение результата
`старт()` выводится само.

Чтобы явно что-то напечатать, есть модуль `ввод_вывод` (доступен только
вне браузера — подробнее дальше в туре и в основной документации):

```panos
импорт ввод_вывод

функ старт() -> Число
    ввод_вывод.печать("привет, panos")
    0
конец
```

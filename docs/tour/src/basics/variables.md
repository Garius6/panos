# Переменные и типы

```panos
пер x = 10
пер имя: Строка = "panos"
```

`пер` — единственный способ объявить переменную. Тип можно не писать —
компилятор выводит его из значения справа.

Базовые типы: `Число` (один числовой тип, без разделения на int/float),
`Строка`, `Булево` (`истина`/`ложь`).

```panos
пер a = 10 + 5 * 2      // Число: 20
пер b = a > 12          // Булево: истина
пер c = "pan" + "os"    // Строка: "panos" — + работает и со строками
```

Сравнение «не равно» — `<>`, не `!=`. Логические операторы — слова, не
символы:

```panos
пер d = истина и ложь        // ложь
пер e = ложь или истина      // истина
пер f = не ложь               // истина
```

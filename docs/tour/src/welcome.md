# Добро пожаловать

Этот тур показывает panos короткими шагами: одна идея на страницу, один
рабочий пример. Проходить лучше по порядку — дальше страницы опираются на
то, что было раньше.

```panos
функ старт() -> Число
    10 + 20
конец
```

Каждый пример в туре можно скопировать в [веб-песочницу](https://garius6.github.io/panos/playground/)
и запустить — ничего ставить не нужно.

Ищете не тур, а подробный справочник или как поставить `panos` локально —
это [основная документация](https://garius6.github.io/panos/).

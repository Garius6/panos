# Contract: Диагностические сообщения

Все сообщения — на русском (FR-015). Формат согласован с существующими:
`Syntactic Error:`, `Semantic Error:`, `Type Error:`, `Runtime Error:`.

## Parser

| Условие | Сообщение |
|---------|-----------|
| Пустой `перечисление` | `Syntactic Error: перечисление '{имя}' должно объявлять хотя бы один вариант` |
| Пустые скобки у варианта | `Syntactic Error: у варианта '{имя}' должны быть либо параметры в скобках, либо скобки должны отсутствовать` |
| Пустое тело `выбор` | `Syntactic Error: выбор должен содержать хотя бы одну ветку` |
| Неподдерживаемый шаблон (литерал/or/охрана) | `Syntactic Error: такой шаблон в выборе пока не поддерживается` |
| Дубликат имени варианта в ADT | `Syntactic Error: вариант '{имя}' объявлен дважды в '{тип}'` |

## Resolver

| Условие | Сообщение |
|---------|-----------|
| Неквалифицированный вариант неоднозначен | `Semantic Error: имя варианта '{имя}' неоднозначно, уточните '{тип}.{имя}'` |
| Квалификатор не является типом | `Semantic Error: '{имя}' — не тип-перечисление, квалификация невозможна` |
| Биндер тени существующего имени | `Semantic Error: имя '{имя}' в шаблоне скрывает переменную/тип с тем же именем` |
| Cross-module форма `модуль.Вариант` неоднозначна | `Semantic Error: имя '{модуль}.{имя}' неоднозначно, уточните '{модуль}.{тип}.{имя}'` |
| `модуль.Тип` — не экспортируемый ADT модуля | `Semantic Error: '{модуль}.{тип}' — не тип-перечисление модуля '{модуль}'` |

## Type checker

| Условие | Сообщение |
|---------|-----------|
| Аргумент конструктора не подходит по типу | `Type Error: у варианта '{тип}.{имя}' поле #{i} ожидает {expected}, получено {actual}` |
| Число аргументов конструктора не совпадает | `Type Error: у варианта '{тип}.{имя}' ожидалось {n} аргументов, получено {m}` |
| subject `выбор` — не sum-тип | `Type Error: выбор ожидает значение перечисления, Опции или Результата, получено {actual}` |
| Непокрытые варианты | `Type Error: выбор не покрывает варианты: {список через запятую}` |
| Недостижимая ветка | `Type Error: недостижимая ветка выбора для '{имя}' в позиции ...` |
| Ветка после `_` | `Type Error: ветка после '_' недостижима` |

## Compiler

Компилятор не должен эмитить ошибок для валидированных программ. Любой
`panic` из этой стадии — внутренняя инвариантная ошибка транслятора,
формулировка: `Compiler Error: внутренняя ошибка при компиляции выбора: {деталь}`.

## VM

| Условие | Сообщение |
|---------|-----------|
| `Match_Fail` | `Runtime Error: значение варианта '{имя_или_?}' не покрыто ни одной веткой выбора` |
| `Get_Variant_Field` промах | `Runtime Error: попытка прочитать поле #{i} у неподходящего варианта` (страховка) |

# Contract: схема `pan.toml` и `pan.lock`

**Feature**: `003-pan-package-manager`

## `pan.toml`

```toml
имя = "мой-пакет"
версия = "0.1.0"
точка_входа = "start.ps"   # опционально, default "start.ps"

[зависимости.имя_пакета]
источник = "https://github.com/panosiki/pan.git"
версия = "^1.2.0"
# алиас = "другое-локальное-имя"   # только при коллизии canonical-имён
```

## `pan.lock`

```toml
версия_формата = 1

[[пакеты]]
имя = "имя_пакета"
источник = "https://github.com/panosiki/pan.git"
разрешённая_версия = "1.3.0"
коммит = "a1b2c3d4e5f6..."
зависит_от = []
```

Соответствует `data-model.md` (Манифест пакета, Lock-файл, Зависимость,
Lock_Запись). Оба файла читаются/пишутся через `std/кодирование/toml.ps`
(`разобрать`/`сериализовать`, либо typed `разобрать_в[T]`/
`сериализовать_из[T]` — на усмотрение реализации в `pan`).

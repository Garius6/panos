## Phase 2: User Story 1 - Правильный автор каждого коммита (Priority: P1) 🎯 MVP

**Goal**: `gitsync sync` подписывает каждый коммит именем РЕАЛЬНОГО автора этой версии хранилища (из отчёта), не единым автором запуска.

**Independent Test**: см. spec.md Independent Test — хранилище с 3 версиями от 2 разных пользователей → `git log --format='%an %ae'` показывает 2 разных автора, совпадающих с записями в `AUTHORS`.
